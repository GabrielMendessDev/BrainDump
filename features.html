Brain Dump v2.0

Bloco de Notas Minimalista com Poderes Incríveis

PDF Import

Importe arquivos PDF e converta automaticamente em notas. O conteúdo é extraído e armazenado como texto.

Temas Dinâmicos

Alterne entre Tema Escuro (padrão) e Tema Claro. Sua preferência é salva automaticamente.

Atalhos Rápidos

Use Ctrl+B, Ctrl+I, Ctrl+U para formatação rápida e Ctrl+S para exportar como .txt.

Export Avançado

Exporte notas como TXT, PDF ou sincronize com pasta local usando Node.js.

Editor Avançado

suporte completo a negrito, itálico, títulos, listas e blocos de código.

Sincronização

Sincronize com pasta local e mantenha backup de todas suas notas de forma segura.

Atalhos de Teclado

Ctrl+B Negrito
Ctrl+I Itálico
Ctrl+U Sublinhado
Ctrl+S Exportar .txt

Começar

PDF Import

1 Clique em ⋮ Menu
2 Selecione Importar PDF
3 Escolha um PDF no seu computador
4 Pronto! Uma nova nota foi criada

Alterar Tema

1 Clique em ⋮ Menu
2 Selecione Tema Escuro ou Tema Claro
3 A interface muda instantaneamente

Sincronização com Pasta Local

1 Abra terminal na pasta do projeto
2 Execute: node sync-server.js
3 Pasta notas-backup/ será criada
